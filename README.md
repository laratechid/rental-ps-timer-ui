## Overview
**Rental PS Timer UI** is a UI built with React TS

- **Language**: Node v22.16

## Installation

Install dependencies:
   ```sh
   npm install
   ```
Set up environment variables (e.g., `.env` file):
   ```plaintext
   create .env and put your config base on .env.example
   ```

## Run Service
   ```sh
   ## watch mode
   npm run dev

   ## production mode
   npm run build
   npm run preview
   ```
